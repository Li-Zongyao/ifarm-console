### farm-console
后台管理系统前端，基于vue+vuex+vue-router+axios+elementUI开发

#### 搭建环境
1. 安装node.js
    https://nodejs.org/en/download/ 
    
2. 全局安装淘宝cnpm 镜像 (可以不用) 
    npm install -g cnpm --registry=https://registry.npm.taobao.org  
    判断cnpm 是否安装成功  
    cnpm -h  

3. 全局安装vue-cli  
    cnpm install -g vue-cli  
    判断vue-cli 是否安装成功  
    cnpm list 
    
4. 下载项目，在跟目录下下载依赖
    npm install
    
5. 运行项目，根目录下运行
    npm run dev
